# Harry-Potter-Cloak
A cloak made using OpenCV, Python

## PRE-REQUISITES
```bash
sudo pip3 install opencv-python
pip install numpy
```
## HOW TO RUN THE CODE
First run back.py to save a background image
```bash
python3 back.py
```
Then run the main cloak file
```bash
python3 cloak.py
```
*Make sure cloak.py and back.py are in the same folder*
